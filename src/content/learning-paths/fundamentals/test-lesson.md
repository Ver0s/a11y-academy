# Markdown Examples

Here's a showcase of commonly used Markdown syntax:

## Text Formatting

Regular text is written as-is. You can make text **bold** or _italic_. You can also combine them for **_bold italic_**.

## Lists

Unordered list:

- Item 1
- Item 2
    - Nested item
    - Another nested item
- Item 3

Numbered list:

1. First item
2. Second item
3. Third item
